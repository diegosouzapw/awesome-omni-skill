## MCP Configuration

### Standard Format

```json
{
  "mcpServers": {
    "ServiceName MCP": {
      "command": "npx",
      "args": ["-y", "package-name@latest"],
      "env": {
        "API_KEY": "required-env-var"
      }
    }
  }
}
```
