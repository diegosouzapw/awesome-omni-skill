```bash
mkdir -p .claude/skills/<name> && curl -sL "<raw_url>" > .claude/skills/<name>/SKILL.md
```

### Browse interactively
Visit the [OmniSkill Registry](https://omni-skill-registry.omniroute.online) for search, filters, and one-click install commands.
